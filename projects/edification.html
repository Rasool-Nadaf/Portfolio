Edification

An enterprise School Management System connecting students, parents, teachers, and administrators on one platform.

Proprietary — screenshots not shown

Overview

Edification is a large-scale enterprise School ERP application built in UIKit. It manages academic activities, attendance, examinations, communication, transport, and a digital library — spanning every stakeholder in a school, from students to admins.

My role

I worked as part of the iOS engineering team on production modules including Homework, Attendance, Examination, Student Progress, Digital Library, Notifications, Lesson Planning, Transport, Sports, Health, and Student Profile — covering UI development, REST API integration, localization, and offline-first functionality.

Key contributions

  • UI development — new UIKit screens, reusable components, and Auto Layout across screen sizes.
  • REST API integration using URLSession and Codable — GET/POST/PUT, error handling, and response parsing.
  • Multilingual localization across labels, buttons, error messages, alerts, and dynamic content.
  • Offline-first architecture — local caching, background sync, and data availability during poor connectivity.
  • Network reachability monitoring, with retry mechanisms and automatic sync on reconnect.
  • Face Recognition Attendance — Implemented a face recognition-based attendance feature for automated, contactless student attendance marking.
  • Production-level bug fixing across UI, API, navigation, layout, and performance issues.
  • Collaborated with backend developers, QA, and PMs using Jira, Bitbucket, and Git through the full release cycle.

What I learned

Edification was my introduction to enterprise-scale iOS development — working inside an established production codebase, taking features from spec to release, and learning how offline-first and localization decisions ripple through an entire app.
