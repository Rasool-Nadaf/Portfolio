NexusChat

A modern communication platform built completely from scratch using SwiftUI and MVVM — secure messaging, groups, and collaboration at scale.

Proprietary — screenshots not shown

Overview

NexusChat is a communication platform I built as an iOS Developer, entirely from scratch, using SwiftUI and an MVVM architecture designed for scale from day one. It supports secure one-to-one and group messaging, user management, and team collaboration features.

My role

I owned the application's architecture end to end — designing the MVVM structure, Repository Pattern, and Service Layer that everything else was built on top of, then implementing the chat experience, authentication, and reusable UI system myself.

Key features

  • One-to-one & group chat, including message forwarding, pinned messages, and user mentions.
  • Authentication & session management, with a full login and registration flow.
  • User profiles with profile editing and user search.
  • Notice Board for organization-wide announcements and updates.
  • AppRouter + NavigationStack for centralized, predictable navigation.
  • A reusable networking layer (APIManager, APIService, APIEndpoints) built for scale.
  • A full library of reusable components — buttons, text fields, cards, chat bubbles, loading views, and alerts.

What I learned

Building NexusChat from the ground up was my first time owning a large-scale SwiftUI architecture solo — designing the Repository Pattern and Service Layer, keeping state management clean with ObservableObject and Published properties, and making sure the whole thing stayed maintainable as features kept getting added.
