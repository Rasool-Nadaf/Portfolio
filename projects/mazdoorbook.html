MazdoorBook

A production-ready workforce-management app — attendance, employees, payments, and payroll — built completely from scratch in SwiftUI.

Get it on the App Store

Overview

MazdoorBook is a workforce-management application designed for small businesses and contractors to manage attendance, employee records, and payments in one place. I built it entirely on my own — from architecture decisions down to payment gateway integration — as a way to prove out a production-grade SwiftUI + MVVM stack end to end.

Architecture

The app is organized into dedicated modules — Authentication, Attendance, Employees, Payments, Profile, Home, and a Super Admin module for higher-level oversight. Networking runs through a reusable APIService / APIManager / APIEndpoint layer with Codable models and the Repository pattern, keeping the UI layer decoupled from how data is fetched or cached.

Key features

  • Secure authentication with session management and Keychain-backed storage.
  • AppRouter + AppState driving NavigationStack-based routing across the app.
  • Offline-first sync — actions taken without a connection are queued and background-synced automatically once connectivity returns, backed by network reachability monitoring.
  • Razorpay Payment Gateway integration, including payment verification, status handling, PDF receipt generation, and receipt sharing.
  • Multilingual localization across English, Hindi, Kannada, and Marathi.
  • Push notifications, Firebase Core, and Firebase Crashlytics for stability monitoring.
  • Reusable UI components built for consistency across every module.

Status

The app is live on the App Store.
