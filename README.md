# Rasool Nadaf — iOS Developer Portfolio

A multi-page portfolio site built with plain HTML/CSS/JS, ready for GitHub Pages.

## Structure
- `index.html` — Home
- `experience.html` — Work experience, education, skills
- `projects.html` — Projects hub
- `achievements.html` — Milestones
- `projects/mazdoorbook.html`, `projects/vastu-cite.html`, `projects/nexuschat.html`, `projects/edification.html` — Project detail pages
- `assets/` — CSS, JS, images

## Deploy on GitHub Pages
1. Create a new GitHub repo (e.g. `rasool-nadaf.github.io` for a root domain, or any repo name for a project site).
2. Push all files in this folder to the repo root.
3. Go to Settings → Pages → set source to the `main` branch, root folder.
4. Your site will be live at `https://<username>.github.io/<repo-name>/` (or `https://<username>.github.io/` if you used the special repo name).
